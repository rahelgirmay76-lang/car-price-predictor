# 🚗 Car Price Predictor

A Machine Learning web application that predicts car prices based on features like year, mileage, and engine size.

## 🔥 Features
- Data cleaning and preprocessing
- Feature engineering (engine size extraction)
- Random Forest Regression model
- Interactive UI built with Streamlit

## 🚀 Demo
Run locally:
```bash
pip install -r requirements.txt
streamlit run app.py
